# 📱 DeFi Strategy Vault Platform - Mobile App

## 📌 Overview

This is the **mobile application** for the DeFi Strategy Vault Platform.
Built using **React Native**, it enables users to:

* Manage strategies on the go
* Connect crypto wallets
* Execute trades
* View AI insights & portfolio performance

The mobile app provides a **fast, secure, and user-friendly Web3 experience** optimized for smartphones.

---

## 🧱 Project Structure

```
mobile/
│── src/          # Core app logic, navigation, services
│── screens/      # App screens (UI pages)
│── components/   # Reusable UI components
│── wallet/       # Wallet integration & Web3 logic
```

---

## ⚙️ Tech Stack

* **Framework:** React Native
* **Language:** JavaScript
* **Navigation:** React Navigation
* **State Management:** Zustand
* **Web3:** Solana Wallet Adapter / Mobile Wallet SDKs
* **API Integration:** Axios

## 🚀 Getting Started

### 1️⃣ Navigate to mobile app

```bash
cd mobile
```

---

### 2️⃣ Install dependencies

```bash
npm install
```

---

### 3️⃣ Run app

#### Android

```bash
npx react-native run-android
```

#### iOS

```bash
npx react-native run-ios
```

---

### 4️⃣ Start Metro bundler

```bash
npx react-native start
```
